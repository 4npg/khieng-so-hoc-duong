# 03. Poster Design Specifications & Production Prompts

## 1. Design System & Layout Specifications

### Color Palette Tokens
* **Primary Cyber Cyan:** `#00f0ff` (Primary glowing accents, title typography, CTA buttons)
* **Shield Emerald Green:** `#10b981` (Legal compliance badges, verification checkmarks)
* **Warning Amber/Rose:** `#f59e0b` / `#f43f5e` (Scam warning badges, threat indicators)
* **Cyber Deep Navy:** `#0a0d18` (Background base depth)
* **Dark Slate Panel:** `#121829` (Glassmorphism backdrop containers)

### Typography Hierarchy
* **Primary Headline Font:** `Plus Jakarta Sans` (Bold 800 / ExtraBold 900)
* **Body / Legal Text Font:** `Be Vietnam Pro` (Medium 500 / SemiBold 600)
* **Monospace Code / Hotline Tags:** `JetBrains Mono` / `Fira Code`
