## 2. Branding Guidelines
* **Logo Badge:** Shield icon with enclosed electric cyan network nodes.
* **Color Usage Rule:** Background max 70% dark navy (`#0a0d18`), cyan accents 20%, emerald green highlights 10%.
* **Font Consistency:** All assets must use Google Fonts `Plus Jakarta Sans` for headers and `Be Vietnam Pro` for body copy.
